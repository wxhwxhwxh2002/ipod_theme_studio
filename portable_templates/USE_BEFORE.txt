【这个整合包做了什么】

这个整合包的目标，是把原本更偏命令行和开发环境的流程，整理成一个更适合普通用户使用的图形界面工具。

目前整合包内已经包含：

- 内置 Python 运行时与所需依赖，不需要额外安装 Python
- 图形界面工作流，不需要手动敲命令完成大多数美术替换工作
- 支持 iPod nano 6、iPod nano 7 (2012)、iPod nano 7 (2015)
- 支持导入官方固件或社区 IPSW
- 支持解包、浏览、预览、替换、重新打包美术素材
- 支持内置裁剪/缩放，方便把手机壁纸或大图直接裁到目标分辨率
- 支持素材收藏库、搜索、备注、删除、再次复用
- 支持 1888 与 0064/0065 相关格式处理和手动降色流程
- 针对 Windows 环境做了额外兼容处理，尽量减少环境配置问题

【简单使用说明】

1. 启动

解压整个文件夹后，直接双击：

launch_theme_studio_portable.bat

即可启动工具。

2. 基本流程

常见使用方式如下：

1. 导入官方固件，或者导入社区 IPSW
2. 进入素材浏览界面
3. 找到想替换的图标、壁纸或缩略图
4. 选择本地图片或收藏素材进行替换
5. 如有需要，可在工具内直接裁剪、缩放或降色
6. 完成后重新打包导出 IPSW
7. 再使用 iTunes / Finder 等方式刷入设备

3. 注意事项

- 本工具不会负责最后的刷机步骤，只负责生成修改好的 IPSW
- 下载官方固件时需要联网
- 第一次使用时，建议先备份原始固件和自己修改后的 IPSW
- 大量把素材从 0064/0065 升到 1888 可能会让资源分区体积变大，建议适度使用
- 不同素材尺寸不同，替换前请注意目标分辨率
