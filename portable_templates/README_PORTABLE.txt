iPod Theme Studio Portable
==========================

This folder is a no-install portable bundle for the GUI workflow.

How to use:
1. Extract the full folder anywhere on your PC.
2. Double-click "launch_theme_studio_portable.bat".
3. Use the GUI to import official firmware or a community IPSW, browse artwork, replace assets, and repack.

Notes:
- Python and required libraries are bundled in "runtime\python".
- Rust, Cargo, and ARM GCC are not needed for the GUI workflow.
- Downloading official firmware still requires an internet connection.
- Repacked IPSW files are for use with iTunes/Finder restore workflows outside this tool.
